# BuhOne

Вёрстке лендинга для компании **BuhOne**.

## Описание проекта

Проект представляет собой адаптивную страницу по макету из Figma.
Страница содержит:

* шапку с логотипом, контактами и навигацией;
* главный экран с фоновым изображением, заголовком, кнопкой и элементами слайдера;
* секцию обратной связи с контактной информацией и формой;
* попап успешной отправки формы;
* футер.

## Функциональность

* Адаптивная вёрстка для desktop, tablet и mobile.
* Все поля формы обязательны для заполнения.
* При успешной отправке формы появляется попап.
* Попап можно закрыть:

  * по кнопке закрытия;
  * по клику на затемнённый фон;
  * по клавише `Escape`.

## Технологии

* HTML5
* CSS3
* JavaScript
* Flexbox
* CSS Grid
* Media queries
* Локальное подключение шрифтов

## Запуск проекта

Проект не требует сборки.

Чтобы открыть страницу локально, достаточно открыть файл:

```txt
index.html
```
